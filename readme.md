# My Favorite Restaurant List

A simple website for my favorite restaurant introduction  on heroku

## Features

- listing restaurants from restaurant api

- searching restaurants by name or categories
<img width="778" alt="image" src="https://user-images.githubusercontent.com/79748426/156925581-342b5740-a513-4a28-be2d-8c70bbd6056c.png">

- review the full restaurant list at the home page 

- clicking on picture to present detailed information for restaurant
<img width="743" alt="image" src="https://user-images.githubusercontent.com/79748426/156925603-fd0b568e-20f6-45f6-8c7a-827f6aea3d2e.png">

- create a new restaurant card
<img width="779" alt="image" src="https://user-images.githubusercontent.com/79748426/156925625-0f85deab-1fb3-44c7-b852-09de999ade18.png">

- update the info whenever you want

- remove the restaurant

- sort the restaurant list in update ascending , update descending , score descending or category
<img width="204" alt="image" src="https://user-images.githubusercontent.com/79748426/156925657-f8f865d3-1135-41fa-8a24-37dae3e574de.png">
